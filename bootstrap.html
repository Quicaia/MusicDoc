Iniciando com Bootstrap

O que é Bootstrap?

Bootstrap é um framework front-end que facilita o desenvolvimento de sites e aplicações web responsivas.

Por que usar Bootstrap?

Bootstrap oferece uma série de componentes prontos, como botões, formulários e navegação, que ajudam a acelerar o processo de desenvolvimento.
